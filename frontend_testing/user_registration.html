📦 2. Bulk Register (JSON Array)

📌 Bulk upload format — Send a JSON array of user objects. Each object must contain:
first_name, last_name, email, role, birth_date, gender (optional: middle_name, suffix, year_level).
Example: [{"first_name":"Ana","last_name":"Reyes","email":"ana@school.com","role":"student","birth_date":"2005-06-15","gender":"Female","year_level":2}]
